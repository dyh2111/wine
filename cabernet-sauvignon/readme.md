# Cabernet Sauvignon Tasting Notes

## 2026-05-08 — 2018 Heitz Cellar Cabernet Sauvignon

**Nose**

- Balsa
- Extra-ripe dark fruits
- Rubber
- Walnut
- Bourbon
- Medium-ripe fruit, like natural sugar-free strawberry jam
- Bright mineral notes

**Palate**

- Higher tannin
- Dry
- Licorice
- Low sweetness, overpowered by tannins
- Not too spicy
- Smooth
- Drywall / chalk

**Mouthfeel**

- Medium mouthfeel

## 2026-05-24 — 2022 Michael David Earthquake Cabernet Sauvignon

**Nose**

- Ripe red fruits
- Jammy
- Mineral / stone
- Strawberries

**Palate**

- Medium-strong tannins
- Pepper
- Light raisins / dates
- Medium sweetness
- Medium complexity

**Mouthfeel**

- Good mouthfeel
- Smooth

**Overall**

- Great nose
- Not as smooth as the earlier Zinfandel
