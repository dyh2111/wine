# Wine Notes

Personal tasting notes, ratings, and buying references for wines I try, collect, or want to understand better.

This repository is my private/public notebook for tracking how my palate develops over time, especially around:

- Vintage Port
- Tawny Port
- California Cabernet Sauvignon
- Zinfandel
- Late Harvest / off-dry reds
- Bordeaux and other benchmark reds

The goal is not just to record scores, but to understand *why* I like or dislike a bottle.

---

## My Current Palate Map

I tend to prefer wines with:

- Noticeable complexity and layers
- Ripe fruit, jammy/dark berry notes, currant, plum, cherry, raisin, date, fig
- Some sweetness or perceived sweetness, but not simple sugar
- Smooth mouthfeel
- Medium tannins, enough structure but not harsh dryness
- Aromatic depth: spice, mineral, tobacco, chocolate, vanilla, licorice, earth

I am still calibrating my palate across dry reds, late-harvest reds, and fortified wines.

### Wines/styles I am currently exploring

| Category | What I am learning |
|---|---|
| Vintage Port | Aging, structure, sweetness, complexity, producer style |
| Tawny Port | Oxidative character, nuttiness, caramel, balance |
| California Cabernet | Ripeness, tannin, oak, drink-now appeal |
| Zinfandel | Jamminess, spice, body, sweetness perception |
| Late Harvest Zin | Sweetness + complexity balance |
| Bordeaux | Structure, dryness, ageability, comparison to Napa Cab |

## Personal Scoring Guide

|  Score | Meaning                                    |
| -----: | ------------------------------------------ |
| 98–100 | Exceptional / benchmark bottle             |
|  95–97 | Excellent, memorable, would strongly rebuy |
|  92–94 | Very good, enjoyable, clear quality        |
|  89–91 | Good but flawed, simple, or not my style   |
|  85–88 | Drinkable but not worth rebuying           |
|    <85 | Significant flaws or not enjoyable         |

My ratings are personal and palate-based. A technically great wine may score lower for me if it is too dry, thin, tannic, or lacks complexity.

## Buying Criteria

Before buying, I want to understand:

1. Is the wine highly rated by credible reviewers?
2. Does the style match my palate?
3. Is it a good value at the current price?
4. Is it ready to drink now or better to cellar?
5. Does it teach me something new?

For Port especially, prioritize:

* Strong vintages
* Proven producers
* Half bottles when useful for tasting/learning
* Vintage Port and aged Tawny over simple domestic port-style wines
* Complexity, mouthfeel, and aging potential
